SIGNAL CHAIN

Voice & model

API key
Free at console.groq.com. Stored only on this machine.
Microphone
Windows must allow desktop apps to use the microphone.
Accessibility
Lets Murmur press Cmd+V to insert text. System Settings, Privacy & Security, Accessibility: turn Murmur on.
Input Monitoring
Lets the hold key work everywhere. If pressing the keycap in General times out, System Settings, Privacy & Security, Input Monitoring: turn Murmur on, then relaunch.
Keep mic warm
Windows takes a moment to open the microphone, so Murmur holds it open briefly after each dictation to make the next one start instantly.
Why this is a setting: while the mic is warm, the Windows mic-in-use indicator stays lit even though nothing is recorded or sent. Choose Off if you want the indicator dark the instant a dictation ends; the tradeoff is a short delay before each one starts.
Transcription model
Prices are pay-as-you-go rates per hour of speech. On Groq's free tier, every model is $0 within generous daily limits.
Smart formatting
A fast LLM removes filler words and fixes punctuation. Falls back to the raw transcript on any hiccup.
Formatting style
Conversation writes naturally. Vibe coding protects technical terms, identifiers, and commands while you dictate about code.
Formatting level
How much polish. None only fixes spelling and punctuation; High rewrites into finished prose without adding anything.
Formatting model
A dictation is about 150 tokens, so even the priciest option here costs a fraction of a cent per dictation.
Language
Auto detects. Pinning one language is a touch faster and more accurate.

WELCOME

Three steps and you're dictating

  1. Get a free Groq API key
    Takes about a minute. , create a key, paste it here.
  2. Allow the microphone
    Windows Settings, Privacy & security, Microphone: turn on "Let desktop apps access your microphone".
  3. Try it
    Click into any text field, then hold Right Ctrl, say hello, and release. Or press Ctrl + Shift + Space.
